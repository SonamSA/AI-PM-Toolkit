# AI-Assisted Agile Project Management Toolkit

This project demonstrates how Generative AI can support Technical Project Managers in Agile software development environments.

The goal of this toolkit is to explore practical ways AI tools can reduce repetitive work such as documentation, reporting, and backlog organization.

## Use Cases Demonstrated

• Meeting summarization  
• Sprint planning assistance  
• Project risk identification  
• Stakeholder status reporting  

## Motivation

Project managers often spend a large portion of their time preparing meeting minutes, project reports, and documentation.

Generative AI tools can assist with these tasks and allow project managers to focus more on delivery management and team coordination.

## Scenario

This prototype is based on a typical **global ecommerce platform development project** where multiple teams collaborate to deliver new features and maintain platform stability.

## Tools Used

• Generative AI tools (ChatGPT / Copilot)  
• Markdown documentation  
• GitHub version control  
• draw.io for workflow diagrams
